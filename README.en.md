# eepromfs

#### Description
eepromfs，基于EEPROM的简易类文件的数据读写库，方便做动态功能增减时参数管理。增减参数块类似增减文件，不会对已有数据存储带来影响。EEPROM硬件资源充裕的情况下使用。
